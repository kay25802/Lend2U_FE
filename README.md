## Commit/PR Convention

| Type       | 설명                                                    |
| ---------- | ------------------------------------------------------- |
| [FEAT]     | 새로운 기능 추가, 기존의 기능을 요구 사항에 맞추어 수정 |
| [FIX]      | 버그 수정                                               |
| [REFACTOR] | 코드 리팩토링                                           |
| [DOCS]     | 문서 수정                                               |
| [ADD]      | 파일, 부가 가능 추가                                    |
| [TEST]     | 테스트 코드                                             |
| [CHORE]    | 패키지 매니저 수정, 그 외 기타 수정                     |
| [BUILD]    | 빌드 관련 수정                                          |
| [STYLE]    | 코드 스타일, 포맷팅에 대한 수정                         |
